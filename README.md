## ✨ یک نگاه

این سامانه یک نرم‌افزار **تک‌صفحه‌ای (SPA)** و **کاملاً سمت کاربر** است؛ تمام اطلاعات به‌صورت
امن روی مرورگر شما (LocalStorage) ذخیره می‌شود و نیازی به سرور یا پایگاه‌داده ندارد.
رابط کاربری با الهام از طراحی **شیشه‌ی مایع (Liquid Glass)**، سایه‌های نرم و حرفه‌ای،
فونت فارسی **وزیرمتن** و آیکن‌های برداری **Lucide** ساخته شده است.

> 🎨 حالت **روشن** و **تیره** • 📅 تقویم **شمسی (جلالی)** • 🔢 اعداد فارسی •
> 💵 نرخ روز **تتر** • 🧾 خروجی **PDF / Word / چاپ** • 🇮🇷 مطابق قوانین مالیاتی **۱۴۰۵**

## 🚀 راه‌اندازی

### ⚡ نصب کاملاً خودکار روی Ubuntu 24 (پیشنهادی)

یک اسکریپت آماده است که Node.js، وابستگی‌ها و اجرای برنامه را به‌صورت خودکار انجام می‌دهد:

```bash
# دریافت پروژه
git clone https://github.com/W2F-Sa/HesabDari.git
cd HesabDari

# اجرای اسکریپت خودکار
chmod +x setup.sh
./setup.sh           # نصب + اجرای محیط توسعه روی http://localhost:5173
```

حالت‌های دیگر اسکریپت:

```bash
./setup.sh build     # فقط ساخت نسخه‌ی نهایی در پوشه‌ی dist/
./setup.sh serve     # ساخت + اجرای سرور تولید روی http://localhost:4173
```

> اسکریپت در صورت نبود `Node.js`، آن را از مخزن رسمی NodeSource (نسخه‌ی ۲۰) نصب می‌کند.
> برای نصب به دسترسی `sudo` نیاز است.

### 🛠️ راه‌اندازی دستی

```bash
# نصب وابستگی‌ها
npm install

# اجرای محیط توسعه
npm run dev

# ساخت نسخه‌ی نهایی
npm run build

# پیش‌نمایش نسخه‌ی build شده
npm run preview
```

پس از اجرا، آدرس نمایش‌داده‌شده (به‌صورت پیش‌فرض `http://localhost:5173`) را در مرورگر باز کنید.

### 🌐 نسخه‌ی آنلاین (GitHub Pages)

نشانی نسخه‌ی منتشرشده:

> **https://w2f-sa.github.io/HesabDari/**

برای فعال‌سازی (یک‌بار): در مخزن گیت‌هاب وارد **Settings → Pages** شوید و در بخش
*Build and deployment* گزینه‌ی **Source** را روی **GitHub Actions** قرار دهید.
سپس از تب **Actions** گردش‌کار *Deploy to GitHub Pages* را اجرا کنید (یا یک commit تازه push کنید).
فایل گردش‌کار در `.github/workflows/deploy.yml` آماده است.

## 🧱 معماری و فناوری

| لایه | فناوری |
|---|---|
| رابط کاربری | React 18 + TypeScript + TailwindCSS |
| ابزار build | Vite 5 |
| مدیریت وضعیت | Zustand + persist (LocalStorage) |
| مسیریابی | React Router 6 (HashRouter) |
| نمودارها | Recharts |
| آیکن‌ها | Lucide React |
| تقویم شمسی | jalaali-js |
| خروجی PDF | html2canvas-pro + jsPDF |
| خروجی Word | HTML → DOC |

```
src/
├── components/      اجزای رابط کاربری (UI، چیدمان، قالب فاکتور)
├── hooks/           هوک‌ها (نرخ تتر)
├── pages/           صفحات اصلی (۱۷ صفحه)
├── store/           وضعیت سراسری، داده‌های اولیه و توابع محاسباتی
├── types/           تعریف انواع داده
└── utils/           ابزارها (اعداد، تاریخ، فاکتور، خروجی)
```

## 🧾 ماژول فاکتور — بیش از ۵۰ قابلیت

- فاکتور **فروش**، **خرید**، **پیش‌فاکتور** و **برگشت از فروش**
- **۴ قالب حرفه‌ای**: مدرن (رنگی)، کلاسیک، مینیمال و **رسمی مالیاتی (سامانه مودیان)**
- درج خودکار **لوگو**، **مهر و امضا** و اطلاعات کامل شرکت
- شماره‌گذاری خودکار با **پیشوند قابل تنظیم**
- چند ردیف کالا/خدمت با انتخاب سریع از فهرست کالا
- **تخفیف درصدی و مبلغی** در سطح ردیف و کل فاکتور
- محاسبه‌ی **مالیات بر ارزش افزوده** برای هر ردیف
- هزینه‌ی حمل، مبلغ گرد کردن و **مبلغ به حروف**
- انتخاب **واحد پول** (ریال/تومان) و نمایش معادل **تتر**
- ثبت **پرداخت‌های چندمرحله‌ای** و محاسبه‌ی مانده
- وضعیت‌های فاکتور: پیش‌نویس، صادرشده، پرداخت‌شده، جزئی، سررسید گذشته، باطل
- **کپی فاکتور**، ویرایش، حذف و جستجو/فیلتر پیشرفته
- **بروزرسانی خودکار موجودی انبار** هنگام صدور فاکتور فروش/خرید
- شماره‌ی مالیاتی، شناسه/کد ملی و کد اقتصادی خریدار برای فاکتور رسمی
- خروجی **PDF**، **Word** و **چاپ مستقیم** بدون به‌هم‌ریختگی فارسی
- خروجی **CSV** از فهرست فاکتورها

## 📦🧮 انبارداری و حسابداری — بیش از ۱۰۰ قابلیت

### انبارداری
- مدیریت کامل **کالا و خدمات** با کد (SKU)، بارکد، نام لاتین و تصویر
- **دسته‌بندی** رنگی کالاها
- پشتیبانی از **چند انبار** و موقعیت‌های مختلف
- قیمت خرید/فروش، نرخ مالیات و محاسبه‌ی **حاشیه‌ی سود**
- حرکات انبار: **ورود، خروج، انتقال بین انبار، اصلاح موجودی، برگشت و ضایعات**
- محاسبه‌ی **موجودی لحظه‌ای** هر کالا در هر انبار
- **حد سفارش مجدد** و **هشدار موجودی کم/ناموجود**
- گزارش **ارزش‌گذاری موجودی** به قیمت خرید و فروش
- خروجی CSV از کالاها، حرکات و گزارش موجودی

### حسابداری
- **دفتر حساب‌ها (کدینگ)** بر اساس استاندارد ایران (دارایی، بدهی، سرمایه، درآمد، هزینه)
- **اسناد حسابداری دوطرفه** با کنترل **تراز بدهکار/بستانکار**
- ثبت **درآمد، هزینه، دریافت و پرداخت** با دسته‌بندی
- مدیریت **صندوق و حساب‌های بانکی** با محاسبه‌ی مانده
- **مدیریت چک‌ها** (دریافتی/پرداختی) با وضعیت و هشدار سررسید
- **مانده‌گیری طرف حساب‌ها** (بدهکار/بستانکار)
- **صورت سود و زیان**، فروش بر اساس کالا و مشتری، هزینه بر اساس دسته
- مدیریت طرف حساب‌ها (مشتری، تامین‌کننده، کارمند) شخص حقیقی/حقوقی
- داشبورد مدیریتی با نمودارهای زنده و شاخص‌های کلیدی

## 💵 نرخ تتر و ارز
- دریافت **هم‌زمان نرخ لحظه‌ای تتر از ۵ منبع**: **نوبیتکس، تترلند، بیت‌پین، اکسیر و TGJU**

- **مبدل تومان ⇄ تتر**
- امکان **ثبت دستی نرخ** در صورت محدودیت دسترسی شبکه
- بروزرسانی خودکار هر ۱۵ دقیقه و ثبت نرخ روی هر فاکتور در لحظه‌ی صدور
